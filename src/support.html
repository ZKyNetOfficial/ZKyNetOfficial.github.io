ZKyNet operates at the application layer inside the browser, where domain information is visible. You set rules like "route banking.com through a direct connection" or "route streaming.example.com through tunnel B" using a visual interface. The system daemon applies per-app routing at the OS level, and the browser client handles per-domain decisions.
